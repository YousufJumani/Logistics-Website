# IRONWAY Logistics Website

A modern, animated logistics landing page built with plain HTML, CSS, and JavaScript.

## Overview

IRONWAY Logistics is a single-page marketing and operations showcase for a premium logistics brand. The site includes strong visual identity, motion-led sections, and interactive UI components for route visibility and dispatch storytelling.

## Tech Stack

- HTML5
- CSS3 (custom animations, responsive layout)
- Vanilla JavaScript (DOM interactions and UI behavior)

## Features

- Cinematic hero section with animated route visuals
- Service capability grid for freight offerings
- Live-network themed tracking interface
- AI dispatch themed section and CTA flow
- Responsive navigation with mobile menu behavior
- Scroll and counter effects for engagement

## Project Structure

- index.html: Markup and page sections
- style.css: Full styling, layout, effects, and responsive rules
- app.js: Interactions, motion logic, and dynamic UI updates
- screenshot files: Visual previews of the design

## Run Locally

Option 1: Open directly

1. Open index.html in your browser.

Option 2: Serve over local HTTP (recommended)

1. From this folder, run:
   - npx serve .
2. Open the local URL shown in terminal.

## Deployment

This project is static, so it can be deployed instantly on platforms like:

- Vercel
- Netlify
- GitHub Pages
